## 주요 기능

- **로그인/회원가입 및 회원탈퇴 로그아웃**
- **펫 CRUD**: 로그인한 User는 펫시터에 맡길 자신의 반려동물을 등록할 수 있습니다
- **예약 CRUD**: 펫을 등록한 User는 자신이 원하는 펫시터에게 예약을 할 수 있습니다
- **리뷰 CRUD**: 펫시터에게 맡겨본 User는 리뷰를 작성할 수 있습니다
